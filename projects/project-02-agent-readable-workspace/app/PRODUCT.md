# PRODUCT

## What it is
A minimal local knowledge base: import documents, browse them in a list, read
one in detail, and (from P01) ask grounded questions. Everything persists to a
local data directory.

## Who it's for
A single user keeping searchable notes/docs on their machine.

## Features (this project adds the first two)
1. **Document import** — add a document by title + body; it's saved to disk.
2. **Document detail view** — click a document to read its full content.
3. **Persistence** — documents survive an app restart.

## Out of scope (for now)
Auth, multi-user, editing/deleting documents, search ranking, file uploads.
